# Altium source files

These three files belong together and should remain in the same folder:

- `G38TPS40200CC.PrjPcb` — Altium PCB project file
- `G38TPS40200CC.SchDoc` — schematic source
- `G38TPS40200CC.PcbDoc` — PCB layout source

## Verified stackup note

The board record in the uploaded `.PcbDoc` links `Top Layer` directly to `Bottom Layer`. This supports describing the design as a **two-layer copper PCB**.
